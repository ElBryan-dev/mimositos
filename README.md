## 📖 Sobre el proyecto

**Mimositos** es una landing page comercial desarrollada para un negocio familiar de regalos personalizados en Perú. El sitio funciona como una vitrina digital completa: muestra el catálogo de productos por categoría, permite ampliar las imágenes y conecta cada pedido directamente a WhatsApp con un mensaje pre-armado, sin necesidad de carrito de compras ni backend.

El objetivo fue construir un sitio rápido, 100% estático, fácil de actualizar y orientado a conversión por WhatsApp — el canal de venta real del negocio.

## ✨ Características

- 🗂️ **Catálogo filtrable por categoría** — Para Papá, Para Mamá, Rosas Eternas, Personalizados, San Valentín y Para Ellos, con más de 20 productos.
- 🔍 **Galería con lightbox** — click en cualquier producto para verlo en grande sin salir de la página.
- 💬 **Pedido directo por WhatsApp** — cada producto genera automáticamente un mensaje personalizado (`Hola Mimositos ✨, me encantó este detalle y quiero pedirlo: *[producto]*`) y abre el chat listo para enviar.
- 📱 **Botón flotante de WhatsApp** con animación de pulso, siempre visible.
- 🎉 **Secciones promocionales por fecha** — por ejemplo, la sección especial del Día del Padre se oculta automáticamente al pasar la fecha límite, controlado con JavaScript puro.
- 🧾 **Sección de políticas de pedido** clara para el cliente (adelanto, saldo, datos de entrega).
- 👨‍👩‍👧 **Sección "Sobre Nosotros"** con enlaces a redes sociales (TikTok, Instagram, WhatsApp).
- 📱 **Totalmente responsive** — menú hamburguesa y grillas adaptables para mobile.
- 🎨 **Diseño cálido y artesanal** — tipografías Nunito/Quicksand, paleta tierra/durazno y animaciones suaves (fade-in, float, hover).
